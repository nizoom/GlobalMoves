## Goal

Track movement of social / political groups (refugees, military units, immigrant waves) based on live news events and visualize onto 3D map

## Technologies used

### Libraries / APIs

- Mapbox
- Chat GPT JS
- Azure News API

### Markers

- Blender

### Frameworks / Scripts:

- Next JS
- Typescript
- Tailwind CSS
